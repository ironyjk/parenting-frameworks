# Differentiated Instruction — Theoretical Foundation

## Overview

Differentiated Instruction (DI) is a teaching philosophy and set of strategies developed by Carol Ann Tomlinson that systematically addresses the reality that students vary in readiness, interest, and learning profile. Rather than teaching to the middle, DI proactively plans multiple pathways to the same learning goals.

---

## The Differentiation Framework

### What You Differentiate

| Element | Definition |
|---------|-----------|
| **Content** | What students learn and materials to access it |
| **Process** | How students make sense of content (activities, thinking) |
| **Product** | How students demonstrate learning |
| **Learning Environment** | Physical and affective conditions |

### Based on Student Characteristics

| Characteristic | How to Assess |
|---------------|---------------|
| **Readiness** | Pre-assessment, formative assessment, prior work |
| **Interest** | Surveys, conversation, observation |
| **Learning Profile** | Preference inventories, observation, self-report |

---

## The KUD Framework

Before differentiating, clarify what ALL students must achieve:
- **Know** — facts, vocabulary, definitions
- **Understand** — Big Ideas, transferable truths
- **Do** — verifiable skills

**Principle:** KUD stays the SAME for all. The pathway changes.

---

## Key Strategies

### Tiered Assignments
Same KUD, different complexity/scaffolding.
- **Approaching:** Guided organizer, provided sources
- **On-grade:** Analyze sources, write paragraph
- **Advanced:** Multiple sources, evaluate and defend position

**Rules:** All tiers equally engaging, equally respectful, based on data not labels, fluid grouping.

### Learning Contracts
Independence in exchange for accountability. Required tasks + choice tasks + timeline + quality standards.

### Anchor Activities
Meaningful independent tasks for early finishers. NOT busywork.

### Flexible Grouping
No permanent "ability groups." Group by readiness, interest, learning profile, mixed, random, or student-choice depending on purpose.

### Choice Menus (Think-Tac-Toe)
Structured menu of options all addressing the same KUD.

### Compacting
Pre-assess -> document mastery -> replace redundant instruction with enrichment. NOT more of the same, NOT permanent tutoring duty.

### Learning Stations
Physical or virtual locations with different tasks. At least one teacher-led.

## Pre-Assessment: The Foundation

| Method | Speed | Depth |
|--------|-------|-------|
| KWL chart | Fast | Surface |
| Entrance ticket | Fast | Targeted |
| Pre-test | Medium | Comprehensive |
| Conference | Slow | Deep |

---

## Managing a Differentiated Classroom

- Teach routines explicitly at year start
- Differentiated classrooms are active, not silent
- Anchor activities absorb time differences
- "Fair means everyone gets what they NEED" — establish this norm early

## What DI Is NOT

| DI Is NOT | DI IS |
|-----------|-------|
| Individualized instruction for every student | Responsive teaching with flexible options |
| Chaotic | Structured with clear routines |
| Lower standards | Same standards, different pathways |
| Permanent ability grouping | Flexible grouping that changes |
| More work for fast finishers | Different, more complex work |
