# Retrieval Practice + Spaced Repetition Skill

> "The act of retrieving knowledge from memory strengthens that memory more than re-studying the same material." — Brown, Roediger, & McDaniel

> Background and theory: Read references/foundation.md

---

## Execution Protocol

### For Educators

**Before the Lesson:**
- Pre-test on upcoming material (primes curiosity)
- Retrieval warm-up: "Write 3 things you remember from last class" (2 min)

**During the Lesson:**
- Pause every 15-20 minutes: "Close your notes. What were the 3 main points?"
- Think-pair-share with retrieval
- Clicker questions / polls

**After the Lesson:**
- Brain dump: write everything remembered (free recall)
- Summary from memory (no notes)
- Retrieval grid: partially completed concept map from memory

**Across the Course:**
- Cumulative low-stakes quizzes (include questions from weeks ago)
- Spaced homework: problems from 2-3 weeks ago alongside current
- Retrieval journals

### For Self-Learners

**Daily:**
1. Before opening notes, spend 5 min writing what you remember
2. Study new material (25-30 min)
3. Close book, write summary from memory
4. Compare to source — identify gaps
5. Create flashcards for gaps (not everything)

**Weekly:**
1. Free recall everything from past week (15 min)
2. Review flashcards due today (spaced repetition)
3. Mixed questions from multiple weeks

**Monthly:**
1. Cumulative practice test
2. Identify "forgotten" topics — they need retrieval, not re-reading
3. Adjust spacing intervals

### For Parents (Study Coaching)

| Situation | What to do instead of re-reading |
|---|---|
| Studying for a test | Practice tests under timed conditions |
| Can't remember material | Free recall first, then check — the struggle IS learning |
| Too much to memorize | Flashcards with spacing (1-3-7-21-60 days) |
| Everything feels familiar | Familiarity is not knowledge — can they retrieve it without looking? |
| Cramming the night before | Redistribute the same time over multiple shorter sessions |

## Key Techniques Quick Reference

| Technique | How | When |
|-----------|-----|------|
| **Free Recall** | Close book, write everything remembered | After each study session |
| **Practice Testing** | Answer questions before and after studying | Daily |
| **Flashcards** | One idea per card, "why/how" not just "what" | Spaced repetition schedule |
| **Interleaving** | Mix topics: ABCBCA not AAABBB | After initial learning |
| **Elaborative Interrogation** | "Why is this true?" "How does this connect?" | During study |

---

## Anti-patterns

1. **Passive re-reading** — highlighting, copying notes, watching videos on repeat
2. **Cramming** — good for tomorrow, terrible for next month
3. **Fluency illusion** — feeling familiar is not the same as knowing
4. **Blocked practice only** — one topic exhaustively before the next
5. **Abandoning retrieval because it feels hard** — the difficulty IS the learning
6. **Flashcard overload** — making 500 cards and never reviewing
7. **Skipping feedback** — retrieval without correction can reinforce errors
