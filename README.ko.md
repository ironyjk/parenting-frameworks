## 왜 만들었나

부모와 교사는 각기 다른 접근이 필요한 수백 가지 상황에 직면합니다. 유아의 떼쓰기와 사춘기 아이의 소통 거부는 완전히 다른 전략이 필요합니다. ADHD 아이와 시험 불안을 겪는 아이에게는 다른 관점이 필���합니다.

하나의 프레임워크로 모든 것을 다룰 수 없습니다. 하지만 올바른 프레임워크를 선택하려면 대부분의 사람들이 가지고 있지 않은 전문 지식이 필��합니다.

**Parenting Frameworks가 이 문제를 해결합니다.** 지능형 라우팅 레이어로서 상황을 설명하면 적절한 프레임워크를 자동으로 선택하고, 순서를 정하고, 통합합니다.

## 빠른 시작

```
/parenting 4살 아이가 놀이터에서 나올 때마다 떼를 써요
```

시스템이 수행하는 작업:
1. 감지: 4세, 훈육/감정 상황, 높은 감정 수준
2. 선택: 감정 코칭(주 프레��워크) + 긍정적 훈육(보조)
3. 생성: 대화 스크립트가 포함된 단계별 전략

### 라우팅 작동 방식

1. **감지 매트릭스** -- 상황의 키워드와 패턴을 프레임워크 시그니처와 매칭 (예: "떼쓴다"는 감정 코칭 + 긍정적 훈육으로 매핑)
2. **연령 필터** -- 아이 나이에 부적합한 프레임워크 제외 (예: 인출 연습은 0-2세에 비적용)
3. **역할 필터** -- 부모, 교사, 양육자 역할별 ��심 프레임워크 풀이 다름
4. **감정 수준** -- 위기 상황은 프레임워크를 우회; 높은 감정 상황은 감정 코칭과 적극적 경청으로 제한
5. **파이프라인 설계** -- 다중 프레임워크 상황은 순서화 (감정 먼저, 행동 다음, 구조 마지막)
6. **충돌 해결** -- 프레임워크 간 충돌 시 우선순위 규칙 적용 (감정 > 행동, 관계 > 훈육, 발달 > 기대)

## 사용 예시

### 예시 1: 유아 떼쓰기 (3세)

```
/parenting 3살 아이가 안 된다고 하면 소리 지르고 물건을 던져요
```

**라우터 선택:** ��정 코칭(주) + ��정적 훈육(보조) + CPS(후속)

**파이프라인:**
- 1단계 (즉시): 감정 코칭 -- "지금 정말 화가 났구나. 화가 나도 괜찮아."
- 2단계 (진정 후): 긍정적 훈육 -- "화가 나는 건 괜찮아. 그런데 물건을 던지는 건 안 돼."
- 3단계 (이후): CPS -- 반복되면 지체 기술을 파악하고 협력적으로 해결

### 예시 2: 숙제 거부 (10세)

```
/parenting 10살 아이가 숙제를 싫어하고 매일 밤 싸워요
```

**라우�� 선택:** 자기결정이론 양��(주) + 성장 ��인드셋(보조) + 인출 연습(지원)

**파이프라인:**
- 진단: SDT -- 자율성이 억압되고 있나? 유능감이 부족한가? 교사와의 관계 문제인가?
- 마인드셋: 성장 마인드셋 -- "나는 수학 못해"에서 "아직 잘 못하지만 나아지고 있어"로 전환
- 방법: 인출 연�� -- 비효율적인 다시 읽기를 능동적 회상과 분산 연습으로 대체

### 예시 3: 사춘기 소통 단절 (14세)

```
/parenting 14살 아이가 대화를 안 해요. 그냥 끙끙대고 방으로 들어가버려요.
```

**라우터 선택:** PET(주) + 자기결정이론 양육(보조) + CPS(지원)

**파���프라인:**
- 경청: PET -- 조언 없는 적극적 경청. 문제 소유권 판별.
- 자율성: SDT -- 통제 줄이기. 선택지 확대. 명령 대신 이유 설명.
- 협상: CPS -- Plan B로 가정 규칙을 함께 만들기.

### 예시 4: 학습 격차가 큰 교실 (교사)

```
/parenting 3학년 반인데 학생들 읽기 수준이 유치원부터 5학년까지 다양해요
```

**라우터 선택:** UDL(주) + 차별화 수업(보조) + 형성평가(지원)

**파이프라인:**
- 설계: UDL -- 처음부터 다양한 표현과 참여 수단 설계. 장벽 제거.
- 실행: 차별화 수업 -- 준비도별 내용, 과정, 결과물 차별화.
- 모니터링: 형성평가 -- 퇴장 티켓, 신호등, 동료 교수로 실시간 이해도 점검.

## 한국 문화 적응 레이어

이 시스템은 한국 교육 환경의 고유한 맥락을 고려합니다:

- **교육열** -- 학원 문화와 선행 학습 현실 속에서 SDT로 "왜 공부하는가"에 초점
- **체벌 금지법 (2021)** -- 체벌 없는 훈육의 구체적 대안 제시 (긍정적 훈육, PET, CPS)
- **학벌 문화** -- 성장 마인드셋으로 과정 칭찬 전환 + 현실적 진로 다양성
- **조부모 양육** -- 핵심 원칙 합의 (감정 수용, 일관성) + 각자 스타일 존중
- **맞벌이** -- 짧더라도 질 높은 감정 코칭 시간. 5분 아침 루틴, 취침 대화, 주말 1:1
- **사교육비** -- 인출 연습 + 성장 마인드셋으로 학습 효율 극대화
