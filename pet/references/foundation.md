# Parent Effectiveness Training (P.E.T.) — Theoretical Foundation

## Overview

Parent Effectiveness Training (P.E.T.) was developed by clinical psychologist Thomas Gordon in the 1960s and has been taught to over 5 million parents worldwide. Its core innovation is the **Problem Ownership** model — determining whose problem it is before choosing a communication tool — combined with **Active Listening**, **I-Messages**, and the **No-Lose Method** (Method III) for conflict resolution.

Gordon rejected both authoritarian power (parent wins, child loses) and permissive surrender (child wins, parent loses). P.E.T. offers a third path: collaborative problem-solving where no one loses.

## The Behavior Window

```
┌─────────────────────────────┐
│   Behaviors I find           │
│   ACCEPTABLE                 │  ← No problem area
│                              │     (relationship-building zone)
├─────────────────────────────┤
│   Child owns the problem     │  ← Child is troubled — use ACTIVE LISTENING
│   (child is bothered)        │
├─────────────────────────────┤
│   Parent owns the problem    │  ← Parent is bothered — use I-MESSAGES
│   (parent is bothered)       │
├─────────────────────────────┤
│   Both own the problem       │  ← Needs conflict — use NO-LOSE METHOD
│   (needs conflict)           │
└─────────────────────────────┘
```

## Gordon's 12 Communication Roadblocks

Responses that BLOCK a child from feeling heard:

1. **Ordering/Commanding** — "Stop crying right now"
2. **Warning/Threatening** — "If you do that again..."
3. **Moralizing/Preaching** — "You should know better"
4. **Advising/Giving Solutions** — "Why don't you just..."
5. **Lecturing/Teaching** — "Let me tell you something..."
6. **Judging/Criticizing** — "That was a dumb thing to do"
7. **Praising/Agreeing** — "You're so smart, you'll figure it out" (yes, even praise can block)
8. **Name-calling/Ridiculing** — "Don't be a baby"
9. **Interpreting/Analyzing** — "You're just jealous"
10. **Reassuring/Sympathizing** — "It'll be fine, don't worry"
11. **Probing/Questioning** — "Why did you do that? What happened first?"
12. **Withdrawing/Distracting** — "Let's talk about something else"

These are especially harmful when the **child owns the problem** and needs Active Listening.

## The Three Methods

| Method | Description | Result |
|---|---|---|
| Method I | Parent wins, child loses | Resentment, rebellion, withdrawal |
| Method II | Child wins, parent loses | Spoiled, anxious child; resentful parent |
| **Method III** | **No one loses — collaborative solution** | **Both needs met, relationship intact** |

---

## When P.E.T. Feels Stuck

1. **Active Listening feels fake at first.** Start with low-stakes situations.
2. **I-Messages don't "work."** They're not designed to control the child — they express YOU honestly.
3. **Method III takes too long.** The first few times are slow. With practice, 5 minutes.
4. **Values collisions** (hair, music, friends). Gordon distinguishes needs conflicts from values conflicts. For values: model, consult (don't impose), accept the child is a separate person.
